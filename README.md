# Astrodata API Client

## Installation

```shell
git clone https://github.com/kenjitheman/astrodata-api-client
```

- Install all dependencies

```shell
yarn install
```

## Usage

- To run:

```shell
yarn start
```

- To run in dev mode:

```sh
yarn dev
```
